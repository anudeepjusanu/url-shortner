## Troubleshooting

### Preview Tab Not Showing?
- Make sure you've created a bio page first
- The Preview tab only appears after you click "Create Bio Page"

### Public URL Not Working?
- Make sure your frontend is running on port 3000
- Check that the backend is running on port 3015
- Verify your username is correct (@kristo)

### Empty Sections?
- "No links added yet" - Add links in the Editor tab
- "No social links added yet" - Add social URLs in the Editor tab
- "No bio added" - Add a bio description in the Editor tab
