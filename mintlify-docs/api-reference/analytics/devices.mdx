title: 'Device Analytics'
api: 'GET https://snip.sa/api/analytics/{urlId}/devices'
description: 'Get device and browser analytics'
---

## Path Parameters

<ParamField path="urlId" type="string" required>
  URL ID
</ParamField>

<RequestExample>

```bash cURL
curl -X GET https://snip.sa/api/analytics/507f1f77bcf86cd799439011/devices \
  -H "X-API-Key: your_api_key_here"
```

```javascript JavaScript
const response = await axios.get(
  'https://snip.sa/api/analytics/507f1f77bcf86cd799439011/devices',
  {
    headers: { 'X-API-Key': 'your_api_key_here' }
  }
);
```

</RequestExample>
