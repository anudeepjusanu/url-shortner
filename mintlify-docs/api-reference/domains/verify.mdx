---
title: 'Verify Domain'
api: 'POST https://snip.sa/api/domains/{id}/verify'
description: 'Verify DNS configuration for a custom domain'
---

## Path Parameters

<ParamField path="id" type="string" required>
  Domain ID
</ParamField>

<RequestExample>

```bash cURL
curl -X POST https://snip.sa/api/domains/507f1f77bcf86cd799439015/verify \
  -H "X-API-Key: your_api_key_here"
```

```javascript JavaScript
const response = await axios.post(
  'https://snip.sa/api/domains/507f1f77bcf86cd799439015/verify',
  {},
  {
    headers: { 'X-API-Key': 'your_api_key_here' }
  }
);
```

</RequestExample>

<ResponseExample>

```json Response
{
  "success": true,
  "message": "Domain verified successfully",
  "data": {
    "domain": {
      "_id": "507f1f77bcf86cd799439015",
      "fullDomain": "go.yourdomain.com",
      "status": "active",
      "verified": true
    }
  }
}
```

</ResponseExample>
