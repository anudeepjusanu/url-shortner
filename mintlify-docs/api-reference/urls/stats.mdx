---
title: 'Get URL Statistics'
api: 'GET https://snip.sa/api/urls/stats'
description: 'Get aggregated statistics for all your URLs'
---

<RequestExample>

```bash cURL
curl -X GET https://snip.sa/api/urls/stats \
  -H "X-API-Key: your_api_key_here"
```

```javascript JavaScript
const response = await axios.get('https://snip.sa/api/urls/stats', {
  headers: {
    'X-API-Key': 'your_api_key_here'
  }
});
```

```python Python
response = requests.get(
    'https://snip.sa/api/urls/stats',
    headers={'X-API-Key': 'your_api_key_here'}
)
```

</RequestExample>
