---
title: 'Get Available Domains'
api: 'GET https://snip.sa/api/urls/domains/available'
description: 'Get list of domains available for URL shortening'
---

<RequestExample>

```bash cURL
curl -X GET https://snip.sa/api/urls/domains/available \
  -H "X-API-Key: your_api_key_here"
```

```javascript JavaScript
const response = await axios.get(
  'https://snip.sa/api/urls/domains/available',
  {
    headers: {
      'X-API-Key': 'your_api_key_here'
    }
  }
);
```

```python Python
response = requests.get(
    'https://snip.sa/api/urls/domains/available',
    headers={'X-API-Key': 'your_api_key_here'}
)
```

</RequestExample>
